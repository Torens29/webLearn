## О проекте
Репозиторий создан для отработки навыков фронтенд-разработки.  
На основе структуры и названий коммитов можно выделить следующие темы:
- Создание и стилизация DOM-элементов (`СозданиеЭлематаИЗзаданиеЕгоСтилей.js`).
- Работа с объектами и методами (`add method it obj`).
- Выполнение HTTP-запросов (`request.js`).
- Вёрстка страниц (`Зад№7`).
- Эксперименты с основами JavaScript.

> Файлы сгруппированы по темам и номерам заданий. Код содержит комментарии и учебные пометки.

## Технологии
- **HTML5** — структура веб-страниц.
- **JavaScript (ES5/ES6)** — логика, манипуляции с DOM, запросы.
- **CSS3** — стилизация и адаптивная вёрстка.
- **Браузерное окружение** — все примеры запускаются в браузере.

## Структура проекта
```
webLearn/
├── request.js # Примеры выполнения HTTP-запросов
├── СозданиеЭлематаИЗзаданиеЕгоСтилей.js # Работа с созданием элементов
├── Зад№7/ # Папка с заданием №7 (вероятно, вёрстка)
├── first/ # Папка с первыми упражнениями
│ └── (файлы с методами объектов)
├── .gitattributes # Настройки Git
└── README.md # Описание репозитория
```
